How to use this:
  • Create a js file with the pack
  • Put the js file in this directory and change the import below to import it instead of another one
  • Open this page in a browser using http-server or similar
  • Open the JS console & see the stuff that's logged
  • Use that info to fine-tune the pack, e.g. by removing any cities that are too close to each other
  • Once it's fine-tuned, copy the output of JSON.stringify(cityData) and create a file with that to add to /dist
